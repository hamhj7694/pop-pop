# 보상과 수집 규칙

## 핵심 원칙

보상은 터뜨리는 흐름을 끊지 않아야 한다. 즉시 모달이나 팝업을 띄우지 않고, 터진 자리에서 작게 등장해 떨어지거나 화면 하단에 쌓이도록 연출한다.

## 랜덤 보상

보상 종류는 다음 세 등급을 기본으로 한다.

- `common`: 짧은 문구, 작은 아이템, 별조각, 스티커, 가벼운 이미지 카드
- `rare`: 특별 이미지, GIF, 희귀 카드, 테마 조각, 효과 조각
- `super_rare`: 기프티콘, 협찬 쿠폰, 경품 응모권, 한정판 아이템, 시즌 아이템

보상 등장 로직은 다음 원칙을 따른다.

- 보상 확률은 입력 흐름을 방해하지 않는 비동기 상태 업데이트로 처리한다.
- 희귀도별 확률은 한 곳에서 관리해 조정 가능하게 둔다.
- 피버타임, 콤보, 이벤트 등 확률 보정은 명시적인 modifier로 분리한다.
- 협찬/경품은 광고처럼 보이지 않고 보상처럼 느껴져야 한다.

## 보상 드랍 연출

- 보상은 작게 등장한다.
- 터진 위치에서 자연스럽게 낙하한다.
- 바닥 또는 수집함 근처에 쌓인다.
- 사용자의 연속 터뜨리기를 방해하지 않는다.
- 문구 보상과 보상 발견 안내는 콤보 알림과 같은 공용 상단 알림 스택에 쌓아 겹치지 않게 한다.
- 보상 알림은 입력을 막는 모달이나 전체 화면 팝업으로 만들지 않는다.
- 희귀도에 따라 낙하 이펙트, 사운드, 진동을 차등 적용할 수 있다.

## 오늘 주운 것들

현재 세션 또는 하루 동안 획득한 보상을 확인하는 임시 수집함이다.

필수 기능은 다음과 같다.

- 하단 고정 버튼 `오늘 주운 것들 N개`
- 획득한 보상 목록 확인
- 희귀도별 필터
- 새로 얻은 보상 강조
- 보상 상세 보기
- 도감 저장

이 버튼과 오버레이는 플레이 흐름을 막지 않아야 하며, 사용자가 직접 열 때만 상세 내용을 보여준다.

## 도감

도감은 장기 수집 시스템이다. 한 번 터뜨리고 끝나는 서비스가 아니라 다시 방문할 이유를 제공한다.

필수 기능은 다음과 같다.

- 전체 수집률
- 일반/희귀/초희귀 분류
- 미획득 콘텐츠는 `???`로 표시
- 획득 날짜 표시
- 최근 획득 항목 표시
- 시즌별 도감 분리 가능성 고려

## 콤보

콤보는 경쟁 요소가 아니라 축하 요소다.

규칙은 다음과 같다.

- 버블을 터뜨릴 때 콤보가 1 증가한다.
- 일정 시간 입력이 없으면 콤보가 종료된다.
- 콤보 종료 시 결과를 보여준다.
- 실패 페널티는 없다.
- `실패`보다 `기록 달성`, `오늘 가장 긴 콤보` 같은 표현을 사용한다.

콤보 단계는 다음과 같다.

- 10 Combo
- 50 Combo
- 100 Combo
- 300 Combo
- 500 Combo
- 1000 Combo

콤보 보상은 PRD 기준을 따른다.

- 10 Combo: 일반 보상 드랍
- 50 Combo: 일반 보상 2개 드랍 가능
- 100 Combo: 희귀 확률 증가
- 300 Combo: 희귀 상자 드랍 가능
- 500 Combo: 희귀 보상 확정 드랍
- 1000 Combo: 특별 보상 또는 한정 아이템 드랍

## 피버타임

피버타임은 낮은 확률로 자동 발동되는 특별 모드다.

기본 규칙은 다음과 같다.

- 일반 터뜨리기 중 낮은 확률로 자동 발동한다.
- 콤보 보상이나 특정 아이템으로도 발동 가능하다.
- 기본 지속 시간은 10초다.
- 보상 등장 확률, 희귀 보상 확률, 터짐 이펙트, 배경, 사운드, 진동이 강화될 수 있다.
- 피버 전용 보상을 별도 카테고리로 관리할 수 있다.

연출 요소는 다음과 같다.

- 화면 밝기 상승
- 배경 반짝임
- `FEVER TIME` 텍스트
- 버블 테두리 발광
- 파티클 증가
- 특수 효과음
- 짧은 진동 패턴
- 남은 시간 표시

피버타임은 사용자가 즉시 알아차릴 수 있어야 하지만 피로하지 않게 짧고 기분 좋게 유지한다.
